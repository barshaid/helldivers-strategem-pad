HELLDIVERS 2 STRATAGEM PAD - FINAL RELEASE
===========================================

FILES:
------

1. HelldiversStrategemPad.apk
   - Android app for your phone
   - Install on Android device (enable "Install from unknown sources" if needed)
   - Make sure phone and PC are on the same Wi-Fi network

2. helldivers_pad_gui.exe
   - Windows PC helper with GUI window
   - Double-click to run
   - Shows a window with server status
   - RECOMMENDED for most users

3. helldivers_pad_server.exe
   - Windows PC helper (console version)
   - Double-click to run
   - Shows server logs in a console window
   - Use if you prefer console output

SETUP:
------

1. Run one of the EXE files on your PC (GUI version recommended)
2. Note your PC's IP address (shown in the window or check ipconfig)
3. Install the APK on your Android phone
4. Open the app - it should connect automatically
5. Make sure both devices are on the same Wi-Fi network

USAGE:
------

- Tap arrow buttons on phone to send WASD keys to PC
- Tap yellow skull button to toggle Left Ctrl
- Use in Helldivers 2 while the game is focused on your PC

NOTE: The app may need administrator privileges to send keypresses.
If keys don't work, right-click the EXE and select "Run as administrator".
